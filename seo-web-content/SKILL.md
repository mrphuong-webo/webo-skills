# SKILL: SEO Web Content Writing

## DESCRIPTION
Write high-quality, SEO-optimized web content that ranks on search engines and converts readers into customers.

## INTENT
CONTENT, SEO, WRITE

## RULES

### 1. Keyword Optimization
- Identify 1 primary keyword and 2–5 secondary keywords
- Place primary keyword in:
  - Title (H1)
  - First 100 words
  - At least 1 subheading (H2/H3)
- Maintain natural keyword density
- Use semantic keywords

### 2. Content Structure
- H1: Title
- H2: Sections
- H3: Details
- Paragraph: 2–4 lines
- Use bullet points
- Add CTA

### 3. Readability
- Simple language
- Active voice
- Conversational tone
- Mobile-friendly

### 4. SEO Best Practices
- Title: 50–60 chars
- Meta: 140–160 chars
- Keyword in slug
- Add FAQ if needed

### 5. Tone
- Clear, confident
- Problem-solving
- No fluff

### 6. Content Depth
- Cover full intent
- Actionable
- Valuable

## OUTPUT FORMAT
- Title
- Intro
- Body (H2, H3)
- Conclusion
- CTA
- FAQ (optional)
